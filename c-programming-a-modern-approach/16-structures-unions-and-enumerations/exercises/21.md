# Exercise 21

What are the integer values of the enumeration constants in each of the following declarations?

```c
enum {NUL, SOH, STX, ETX};
```

- 0, 1, 2, 3

```c
enum {VT = 11, FF, CR};
```

- 11, 12, 13

```c
enum {SO = 14, SI, DLE, CAN = 24, EM};
```

- 14, 15, 16, 24, 25

```c
enum {ENQ = 45, ACK, BEL, LF = 37, ETB, ESC};
```

- 45, 46, 47, 37, 38, 39
